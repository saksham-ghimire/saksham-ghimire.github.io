Employee Management System

  1. Projects
  2. Employee Management System

Description

The Employee Management System is a robust cross-platform solution employing Golang for client-side and Python for server-side applications. The client, deployed on individual computers, systematically communicates with the server, residing on a centralized system. Periodic updates sent by the client include essential system metrics like CPU usage, network traffic, and active applications. Utilizing gRPC for communication ensures a streamlined and reliable exchange of data between client and server.

Upon receipt, the server processes and stores this data in a dedicated database, forming the foundation for comprehensive analysis and reporting. Noteworthy features encompass visualization tools and report generation, empowering administrators with valuable insights into workplace activities.

The client application, enhanced by Win32 API integration, interfaces with Windows processes, enabling advanced functionalities. Notably, the system can enforce IP blocking by adding rules to the Windows firewall, bolstering network security. This multifaceted Employee Management System is designed to monitor, analyze, and optimize workplace environments. Its features make it valuable for scenarios such as proactive network management, security control, and resource optimization in diverse organizational settings.

Note: This was done as a final year project, so code might optionally need changes and may not be production ready.

Stacks Used

  • Golang: Used to create client application that communicated with winAPI on windows and file system on linux, to provide advanced feature like IP blocking and health diagnosis
  • Python: Provides a centralized system that periodically receives updates from client, stores for visualization and report generation purposes, also provides a user interface
  • Javascript: Uses javascript (Chart JS) for graph visualization providing advanced analytics
  • Elasticsearch: Used as a primary source for data storage
