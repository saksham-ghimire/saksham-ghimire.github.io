Description

In the realm of API development, a comprehensive suite of interfaces has been meticulously crafted to address specific needs. A central element is the facade API, strategically designed to interact with various internal APIs, providing a unified and coherent response. This facade API plays a pivotal role in orchestrating communication with both SQL and NoSQL databases, following a hybrid model for an optimized architectural overview. To streamline intercommunication and foster easy integration, specialized packages were developed. These packages serve as connectors between APIs, preventing redundant work and facilitating seamless interactions within the system. Active participation in architectural overviews and flowchart creation has been a key focus, determining the isolated nature of certain APIs to minimize unintended interdependencies in the microservices architecture.

Advanced middleware integration is a hallmark, particularly for JWT validation, ensuring secure and authenticated transactions. An initiative was undertaken for Single Sign-On (SSO) using Keycloak, enhancing user authentication and authorization processes. Furthermore, an advanced logging implementation, including the utilization of trace IDs, has been deployed to meticulously track internal API calls, providing insightful monitoring and diagnostics.

In summary, the APIs developed are not merely standalone entities but components intricately woven into a sophisticated system. The facade API orchestrates seamless communication with internal APIs, following a hybrid database model. Specialized packages optimize intercommunication, while architectural insights and advanced features such as middleware integration, SSO, and logging contribute to the robustness and efficiency of the overall API ecosystem.

Stacks Used

  • Golang: Using GIN framework to facilitate communication between backend and frontend services
  • Python: Using falcon, fast api to facilitate communication between backend and frontend services. Using django for advanced applications requiring individual sessions
