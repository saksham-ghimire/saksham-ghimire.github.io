Description

The developed open-source library efficiently matches JSON rules against streaming data, supporting a variety of operations. Written in Golang, the library emphasizes comprehensive Test-Driven Development (TDD), ensuring reliability. Rule formats, like {"a__eq": 1}, facilitate matching against JSON structures.

Notably, the library supports aggregation features, enabling users to define actions triggered by aggregate occurrences. Events are chained until a user-defined function is invoked. The library extends its capabilities to Python and JavaScript packages, currently under development.

Moreover, the library incorporates advanced propositional logic boolean algebra, allowing users to build intricate rule structures with logical operators such as AND and OR. This feature enhances the library's versatility, enabling sophisticated rule-based operations on streaming data.

In summary, the library offers a versatile solution for real-time data processing with rule-based matching, supporting aggregation features and advanced propositional logic boolean algebra. With Golang implemented and Python/JavaScript packages in development, the library provides a cross-language, robust tool for handling diverse streaming data scenarios.
